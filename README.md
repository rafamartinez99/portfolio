## Proyectos

**Granalladora APH** — C#, .NET 8, WPF, OPC UA, WCF, Telerik
Suite de monitorización en tiempo real de máquinas de granallado industrial desarrollada en prácticas en Acerinox.

**Agente SFTP** — C#, .NET 8, SSH.NET, AES
Agente de bandeja del sistema que monitoriza carpetas locales y sube archivos automáticamente a servidores SFTP. Desarrollado en prácticas en Acerinox.

**EncryptorACX** — C#, .NET, WPF, AES
Herramienta de escritorio para cifrar credenciales, usada por el equipo técnico junto al Agente SFTP.

**SuperManzanares** — Kotlin, Jetpack Compose, Firebase, Room, Hilt, Mapbox
App Android de supermercado desarrollada como Trabajo de Fin de Grado.

**StarWars Compose** — Kotlin, Jetpack Compose, Retrofit, Coil, Hilt
App Android que consume la API pública SWAPI para explorar personajes y películas de Star Wars.
